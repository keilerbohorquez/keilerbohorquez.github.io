En HTML5 no es necesario proporcionar valores para todos los atributos, como los controles, reproduccion automatica y atributos de bucle utilizados con el elemento <video>. Estos atributos son como interruptores de encendido/apagado. Si el atributo esta presente, activa esa opcion. Si se omite el atributo, la opcion esta desactivada. Si el navegador no admite el elemento <video> o el formato de video utilizado, se mostrara lo que este entre la <video> de apertura y las eqtiquetas de </video> de cierre.

preload: este atributo le indica al navegador que hacer cuando se carga la pagina. puede tener uno de tres valores:
none: el navegador no debe cargar el video hasta que usuario presiona play.
auto: el navegador debe cargar el video cuando se carga la pagina.
